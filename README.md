# Simple Tools & Daily Scripts

This repository contains a collection of small, practical scripts designed to support daily technical tasks, automate repetitive work, and serve as a learning and practice space for scripting, logic, and problem-solving.

The focus is on simplicity, clarity, and usefulness, making these scripts easy to understand, adapt, and extend.

## Purpose

Provide quick solutions for common day-to-day tasks

Practice scripting techniques, logic, and problem solving

Experiment with small ideas without overengineering

Build a personal toolbox of reusable utilities

## Scope

The scripts in this repository may include (but are not limited to):

- File and directory management
- Text processing and data manipulation
- Simple automation tasks
- Utility scripts for development and operations
- Logic and algorithm practice
- Proofs of concept and small experiments

Scripts are intentionally kept small and focused, prioritizing readability over complexity.
